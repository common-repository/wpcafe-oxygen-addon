== Description ==

WPCafe Oxygen Builder Addons enables the [WPCafe](https://themewinter.com/wp-cafe/) Food Menu and Restaurant Reservation plugin to parse WPCafe widgets inside the Oxygen editor for Pages.

WPCafe Oxygen Builder addons provide free Oxygen Builder elements for creating cafe and restaurant websites. If you can use Oxygen builder then you are ready to use WPCafe Oxygen addons for your food ordering business websites.

To use this plugin, you must first have all the plugins installed and activated. The required plugins are:
1. [Oxygen](https://oxygenbuilder.com/)
2. [WPCafe](https://wordpress.org/plugins/wp-cafe/)
3. [WooCommerce](https://wordpress.org/plugins/woocommerce/)
4. [WPCafe Oxygen Addon](https://wordpress.org/plugins/wpcafe-oxygen-addon/)
5. [Oxygen Elements for WooCommerce](https://oxygenbuilder.com/tour/woocommerce/)

**Check [WPCafe PRO](https://wordpress.org/plugins/wp-cafe/) || [Support](https://themewinter.com/support/) || [Documentation](https://support.themewinter.com/docs/plugins/wp-cafe/)**

### Features of WPCafe Oxygen Addons ###

**Food Menu**- You can use food menu elements on oxygen builder addons. List style food menu, tab style food menu and location-based food menu elements available. With the food menu elements, you can add an appealing food menu on your Oxygen website.

**Restaurant Reservation**- Reservation elements will enable you to add a reservation form at your Oxygen site. With the form of reservation, you can take the user's reservation from the website directly.

### About WPCafe ###

[WPCafe](https://themewinter.com/wp-cafe/) is a complete WooCommerce food ordering, pickup and delivery plugin to launch restaurant websites. You can create a food menu, restaurant reservation, food ordering and delivery system (PRO). Moreover, WPCafe comes with WordPress shortcode, Elementor widgets and Gutenberg block to make your Restaurant website simple and easy to use.

= Should I install WPCafe and Oxygen Builder Plugin for the addons? =

YES! The addons work with WPCafe and Oxygen Builder. To get full functionality, you will need to install both of the plugins.
